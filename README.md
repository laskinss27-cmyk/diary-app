# Дневник — приложение для записи чувств

Приватный дневник настроения для Android. Помогает выписать, что внутри, увидеть динамику за неделю и не остаться один на один с тяжёлым днём. На русском, без регистраций, без облака — все записи живут только на твоём телефоне.

> ⚠️ **Альфа-сборка.** Приложение в активной разработке, ещё не опубликовано в Google Play. Установка через APK из релизов.

---

## Что умеет

### Дневник и анализ настроения
- **Свободная запись** — пишешь как удобно, голос тоже принимается (распознавание речи).
- **Оценка от 1 до 10** — каждой записи присваивается балл и эмоциональный ярлык. Цветной круг от красного (тяжело) до зелёного (хорошо).
- **Два режима анализа:**
  - **Офлайн** — встроенный лексиконный анализатор со стемминтом (snowball), стоп-словами и подбором ключевых эмоций. Работает без интернета.
  - **ИИ** — Gemini, если пользователь подключил свой API-ключ. Тонко улавливает контекст и сарказм.
- **Тихий пересмотр** — записи, проанализированные офлайн, автоматически переанализируются ИИ, когда появляется сеть. Балл обновляется только при значимом расхождении.

### Защита и поддержка
- **Кризис-детектор** — ловит маркеры суицидальных мыслей и самоповреждения. Не нравоучает, мягко предлагает помощь.
- **Раздел «Если плохо»** — кризисные горячие линии РФ / РБ / КЗ, подбор по городу, прямые кнопки звонка и WhatsApp.
- **Техники саморегуляции** — анимированное дыхание 4-7-8 и заземление 5-4-3-2-1.
- **Раскраски** — 7 встроенных контурных картинок, заливка по тапу, любая палитра, ластик, пипетка, зум двумя пальцами. Готовую раскраску можно прикрепить к записи.

### Ритм и привычка
- **Ежедневные напоминания** — точные будильники (`exactAllowWhileIdle`) с подсказкой для MIUI/Xiaomi и быстрыми кнопками к настройкам автозапуска и батареи.
- **Достижения** — 7 скрытых ачивок (первая запись, серия 7/30 дней, идеальный день, сова, заземление и др.) с диалогом, картинкой и звуком при получении.
- **Календарь** — все записи на месячной сетке с цветовой подложкой по настроению.
- **Отчёт за неделю** — график динамики, средний балл, топ-эмоции, сравнение с прошлой неделей.
- **Экспорт в PDF** — сводка записей за период.

### Приватность
- Все данные в локальной SQLite на устройстве. Никаких аккаунтов и серверов проекта.
- Если используешь Gemini — ключ хранится только у тебя, обращения идут напрямую с телефона.
- Записи в списке свёрнуты в секции «Сегодня / Вчера / Ранее»: видны только цветной бейдж и время. Текст открывается по тапу — на случай если кто-то заглянет в экран.

## Установка

1. Скачай свежий APK со страницы [Releases](https://github.com/laskinss27-cmyk/diary-app/releases).
2. Разреши установку из неизвестных источников.
3. На MIUI/Xiaomi придётся подтвердить системный диалог установки.
4. Для надёжных уведомлений: разреши автозапуск приложения и сними ограничения батареи (приложение покажет быстрые ссылки в настройках).

После установки приложение само сообщит о новой версии и предложит скачать.

## Стек

- **Flutter 3.11+** / Dart 3
- **SQLite** (`sqflite`) для записей
- **flutter_local_notifications** для напоминаний
- **speech_to_text** для голосового ввода
- **Google Gemini API** (опционально) для ИИ-анализа
- **image** + isolate-based scanline flood fill для раскрасок
- **flutter_colorpicker**, `pdf`, `printing`, `table_calendar`, `audioplayers`

Архитектура: экраны в `lib/screens`, бизнес-логика и данные в `lib/services`, модели в `lib/models`. Никакого тяжёлого state-management — всё на `setState` и `ChangeNotifier`, потому что приложение по большей части линейное.

---

## Сборка из исходников

```bash
flutter pub get
flutter build apk --release
```

Если хочешь использовать ИИ-анализ: скопируй `lib/services/secrets.example.dart` в `secrets.dart` и положи туда свой Gemini API-ключ. Без ключа приложение работает на офлайн-анализаторе.
